CMD+K Terminal AI

Transform natural language into shell commands with AI

What is CMD+K Terminal AI?

A powerful VS Code extension that brings AI-powered command generation directly to your terminal. Press CMD+K (or Ctrl+K) in any terminal, describe what you want to do in plain English, and let AI generate the perfect command for you.

✨ Inspired by Cursor's Terminal Assist feature

Key Features

🚀

Natural Language

Type what you want to do, not how to do it. "find all large files" becomes the right command for your system.

🧠

Multiple AI Providers

Choose from OpenAI (GPT-4), Anthropic (Claude), or xAI (Grok). Switch providers and models on the fly.

🌐

Context-Aware

Automatically detects your OS, shell type, working directory, and command history for accurate suggestions.

👁

Review Before Execute

Generated commands appear in your terminal for review. Edit, execute, or cancel - you're always in control.

⚡

Real-time Streaming

Watch commands generate character-by-character with streaming AI responses.

🔧

Cross-Platform

Works on macOS, Windows, and Linux with support for bash, zsh, PowerShell, fish, and cmd.

Technical Details

Supported AI Providers

  • OpenAI (GPT-4, GPT-4o, GPT-3.5)
  • Anthropic (Claude Sonnet 4.5, Opus 4.1)
  • xAI (Grok-4, Grok-beta, Grok-code)

Context Information

  • Operating System
  • Shell Type & Version
  • Current Working Directory
  • Recent Command History

How It Works

1

Activate

Press CMD+K in any VS Code terminal

2

Describe

Type what you want to do in natural language

3

Review

See the generated command appear in your terminal

4

Execute

Press Enter to run or edit first - you're in control
