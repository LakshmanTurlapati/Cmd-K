   - `DISPLAY` set -> X11
   - Otherwise -> Unknown

   - On Wayland: try `wl-copy` first if available, fall back to xclip
   - On X11: use `xclip -selection clipboard`
   - If neither available: use `arboard::Clipboard` as final fallback
   - Pipe command text to stdin of the clipboard tool

5. Change `paste_to_terminal` return type from `Result<(), String>` to `Result<String, String>`. Update ALL platform paths:
   - macOS path: change final `Ok(())` to `Ok("auto".to_string())`
   - Windows path: change final `Ok(())` to `Ok("auto".to_string())`
   - Linux path: replace the stub with a real implementation

6. Implement `paste_to_terminal_linux()` helper (see RESEARCH.md full X11 paste flow):
   - Read `AppState.linux_tools` and `AppState.previous_app_pid`
   - Call `detect_display_server()`
   - Write command to clipboard via the Linux `write_to_clipboard`
   - If X11 + xdotool available (can auto-paste):
     a. `xdotool search --pid PID` to find window ID (take first line)
     b. `xdotool windowactivate --sync WINDOW_ID` to focus terminal
     c. `xdotool key ctrl+u` to clear current line
     d. `std::thread::sleep(50ms)` for shell processing
     e. `xdotool key ctrl+shift+v` to paste from clipboard
     f. Return `Ok("auto".to_string())`
   - If Wayland or no xdotool: clipboard already written, return `Ok("clipboard_hint".to_string())`

7. Change `confirm_terminal_command` return type from `Result<(), String>` to `Result<String, String>`. Update ALL platform paths:
   - macOS: change `Ok(())` to `Ok("auto".to_string())`
   - Windows: change `Ok(())` to `Ok("auto".to_string())`
   - Linux: replace stub with real implementation

8. Implement `confirm_command_linux()` helper:
   - If X11 + xdotool: `xdotool key Return`, return `Ok("auto".to_string())`
   - Otherwise: return `Ok("confirm_hint".to_string())`

9. Update the Linux cfg block in `paste_to_terminal` to import `AppState` (add `target_os = "linux"` to the existing `#[cfg(any(target_os = "macos", target_os = "windows"))]` import of `AppState` at line 12, changing it to include linux).
