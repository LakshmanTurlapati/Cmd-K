<task type="auto">
  <name>Task 1: Create Windows build script and add npm script wrappers</name>
  <files>scripts/build-windows.sh, package.json</files>
  <action>
1. **Create `scripts/build-windows.sh`** — A simple wrapper for Windows NSIS builds:

   ```bash
   #!/usr/bin/env bash
   # Build Windows NSIS installer for CMD+K
   # Requires: Windows with Rust toolchain, Node.js, pnpm
   # Output: src-tauri/target/release/bundle/nsis/

   set -euo pipefail

   echo "=== CMD+K Windows Build ==="
   echo "Building NSIS installer..."

   # Install frontend dependencies
   pnpm install

   # Build the Tauri app (produces NSIS installer on Windows)
   pnpm tauri build

   echo ""
   echo "=== Build Complete ==="
   echo "Installer location: src-tauri/target/release/bundle/nsis/"
   ls -la src-tauri/target/release/bundle/nsis/*.exe 2>/dev/null || echo "(Run on Windows to see .exe output)"
   ```

   Make it executable: `chmod +x scripts/build-windows.sh`

2. **Add npm scripts to `package.json`** — Add `build:mac` and `build:windows` to the scripts section:

   ```json
   "build:mac": "./scripts/build-dmg.sh",
   "build:windows": "./scripts/build-windows.sh"
   ```

   Add these after the existing `"tauri"` script entry. Keep all existing scripts unchanged.

3. **Commit**:

   - macOS: ./scripts/build-dmg.sh (existing notarized DMG pipeline)
   - Windows: ./scripts/build-windows.sh (NSIS installer via tauri build)"

<task type="auto">
  <name>Task 2: Verify macOS build compiles and delete windows branch</name>
  <files></files>
  <action>
1. **Verify macOS compilation** — Run `pnpm tauri build` to confirm the merged codebase compiles on macOS. This tests that:
   - All `#[cfg(target_os = "windows")]` code is properly excluded on macOS
   - No missing macOS dependencies after merge
   - Frontend builds correctly
   - Tauri bundling succeeds

   ```bash
   pnpm tauri build 2>&1 | tail -30
   ```

   If this fails, diagnose and fix. Common issues:
   - Missing `#[cfg]` gate on a Windows import → add the gate
   - Cargo.lock conflicts → run `cargo generate-lockfile` in src-tauri/
   - Frontend build error → check for import path issues

2. **Delete local windows branch**:
   ```bash
   git branch -d windows
   ```
   Use `-d` (not `-D`) since it's been merged. If git refuses (claims not merged), use `-D` since we know the merge was performed.

3. **Delete remote windows branch**:
   ```bash
   git push origin --delete windows
   ```
