The `use commands::{ ... }` block imports many items. Several are used across both platforms (show_overlay, hide_overlay, register_hotkey, etc.). Keep these unconditional. The key change is gating the external crate imports (tauri_nspanel, NSVisualEffectMaterial).

pub previous_hwnd: Mutex<Option<isize>>,

Initialize in `Default::default()` as `previous_hwnd: Mutex::new(None)`.
