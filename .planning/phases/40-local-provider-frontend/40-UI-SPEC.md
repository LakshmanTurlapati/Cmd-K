### Interaction states

| State | Behavior |
|-------|----------|
| Local provider selected at step 0 | Step 1 (API Key) auto-skipped; stepper dot 2 shows checkmark (completed) |
| Local server not running at step 2 | Empty state shown with "No models found" / "Is your server running?"; "Skip this step" link available |
| Local server running at step 2 | Models fetched and displayed in tiered list (same as cloud providers) |
| Skipped step in stepper | Renders as completed (checkmark icon in `bg-[#48484A]` circle), NOT removed from layout |
